# 麻雀全自動計算アプリ
## 目的：柔軟性のあるシステムを目指し、設計、実装をする。
現在は、Mリーグのルールに沿って役を定義しているが、他の〇〇ルールを入れたい！その役一時的に削りたい！などの要望に対して、複数人で開発しても、コンフリクトが起きなく、素早く開発できるアーキテクトを目指す
Mリーグ公式ページ：https://m-league.jp/about/
